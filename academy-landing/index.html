A course that compounds.

Each phase depends on the last. You start with one Linux machine, then turn that base into automation, delivery, observability, and platform work.

01 Systems

Build a Linux baseline, ship a static site, proxy an app, add a database, and debug it under pressure.

Labs 01-05 02 Automation

Turn manual setup into scripts and Ansible, then manage secrets and multi-VM configuration.

Labs 06-09 03 Delivery

Move from containers to CI, self-hosted runners, registries, progressive delivery, and rollback.

Labs 10-16 04 Operations

Instrument the stack with metrics, logs, traces, SLOs, backups, and failure recovery drills.

Labs 17-22 05 Platform

Finish with Terraform, GitOps, Kubernetes, a delivery platform capstone, and AI-assisted ops.
