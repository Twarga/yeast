## Supported Images

The current image list lives in [Images](images.md).

Use this command to confirm the list from the installed CLI:

```bash
yeast pull --list
```

## Not Supported In v1.1

Do not teach these as public config fields yet:

- `ports`
- `host_port`
- `guest_port`
