Yeast
GitHub Install Quickstart

Turn a folder into real VMs.

Yeast is a Linux-first local VM orchestrator for QEMU/KVM. Describe machines in YAML. Run yeast up. SSH in. Done.

Get Started View on GitHub
terminal

$ mkdir my-lab && cd my-lab

$ yeast init

OK Project initialized

$ yeast pull ubuntu-24.04

$ yeast up

✓ Started web (127.0.0.1:2222)

$ yeast ssh web

yeast@web-lab:~$

What you get

YAML-driven

One yeast.yaml per project. Declare what you want. Yeast makes it real.

cloud-init ready

Users, SSH keys, hostnames, and networks are injected automatically on first boot.

Provisioning

Install packages, copy files, and run shell commands over SSH after the VM boots.

Snapshots

Save a clean baseline. Restore it later. Perfect for lab reset workflows.

Private lab networks

Connect multiple VMs on a static-IP private network. VM-to-VM, isolated from the host.

JSON & events

Every command supports --json. Lifecycle commands stream --events. Built for tools.

Install

curl -fsSL https://raw.githubusercontent.com/Twarga/yeast/main/install.sh | bash

Requires Linux, /dev/kvm, and qemu-system-x86_64.
