# Templates

Templates are project starters.

List built-in templates:

```bash
yeast init --list-templates
```

Create from a built-in template:

```bash
yeast init --template ubuntu-basic
```

Built-in templates:

- `ubuntu-basic`
- `caddy-single-vm`
- `two-vm-lab`

After initialization, the files are copied into your folder and become a normal editable Yeast project.
