# Twarga Academy Modules

Twarga Academy is organized into five course modules. Each module groups several labs so the reader can see progression instead of a long wall of lessons.

## Module 1: Systems

Labs 01-05

- Linux server baseline
- Static site with Nginx
- Reverse proxy to backend app
- Database-backed app
- Linux troubleshooting drill

## Module 2: Automation

Labs 06-09

- Bash automation for server setup
- Ansible for one server
- Ansible for multi-VM web cluster
- Secrets and configuration management

## Module 3: Containers and delivery

Labs 10-16

- Docker fundamentals on a VM
- Compose multi-service app
- Container build, scan, hardening
- CI with GitHub Actions
- Self-hosted CI runner
- Private container registry
- Progressive delivery and rollback

## Module 4: Operations

Labs 17-22

- Prometheus and Grafana monitoring
- Centralized logging
- OpenTelemetry distributed tracing
- SRE: SLOs, alerts, error budgets
- Backup and restore drill
- Chaos and failure recovery drill

## Module 5: IaC, GitOps, platform, AI

Labs 23-30

- Terraform fundamentals
- Terraform modules and environments
- GitOps with Argo CD or Flux
- End-to-end delivery platform
- Kubernetes foundations with k3s
- Kubernetes networking, config, storage
- Kubernetes delivery capstone
- AI-assisted DevOps and local LLM ops
