## Course flow

| Phase | Labs | Focus |
|---|---:|---|
| Systems | 01-05 | Linux, web servers, troubleshooting |
| Automation | 06-09 | Bash, Ansible, secrets |
| Containers and delivery | 10-16 | Docker, Compose, CI, registry, rollout |
| Operations | 17-22 | Monitoring, logging, tracing, SLOs, recovery |
| IaC, GitOps, platform, AI | 23-30 | Terraform, GitOps, Kubernetes, LLM ops |

## Reading rule

Read in order. Each lab assumes the last one worked.
